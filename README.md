# Customer Segmentation

####

**Dados**
> Conjunto de dados transacional que contém todas as vendas ocorridas entre 01/12/2010 e 09/12/2011 para um e-commerce do varejo. Muitos dos seus clientes são atacadistas. 

<br>Fonte: https://archive.ics.uci.edu/ml/datasets/online+retail

<br>

**Objetivo**
> Este projeto tem como objetivo extrair informações relevantes para que sejam utilizadas na construção de campanhas de retenção e fidelização de clientes. Inicialmente através da segmentação de clientes.
