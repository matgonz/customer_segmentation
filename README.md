# Customer Segmentation

####

**Dados**

> Conjunto de dados transacional que contém todas as vendas ocorridas entre 2010 e 2011 de um e-commerce do varejo.

<br>Fonte: https://archive.ics.uci.edu/ml/datasets/online+retail

<br>

**Objetivo**

> Segmentar a base de clientes utilizando análise RFM. Os resultados serão utilizados para criar estratégias de retenção, fidelização e aumento da receita.
>
> Estratégias:
>
> - Quem são os melhores clientes?
> - Quais clientes teriam potencial para comprar mais vezes?
> - Quais clientes já estão perdidos e não merecem mais tanta atenção?
> - Quais clientes eu não posso perder?
> - Quais são os meus clientes mais fiéis?

**Desenvolvimento**

> 1. Aquisição e preparação dos dados
> 2. Segmentação RFM
> 3. Descrição das características de cada segmento
> 4. Análise de métricas de receita (LTV, MRR, Taxa de crescimento da receita), Churn vs Segmentos
